the pipeline

z: router=ollama, model=qwen → z-command override → static routing rules → embedding classifier → decision llm → selected backend → semantic cache → response + tap log

every request flows through four tiers of intelligence. z-commands let you override routing instantly. static rules handle 90% of cases. embeddings classify simple queries. the decision llm makes tough calls. all cached and logged.
