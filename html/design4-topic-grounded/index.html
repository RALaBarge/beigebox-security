How BeigeBox Works

4-Tier Routing Pipeline with Embedding Classifier & Decision LLM

Your request enters BeigeBox's intelligent pipeline. z: commands override routing. Static rules match models to backends. Embedding classifiers handle simple queries. The decision LLM makes complex calls. All while caching semantically and logging everything to Tap.
