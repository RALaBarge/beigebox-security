What You Get

🎯 Smart Routing

4-tier decision pipeline: Z-commands → embedding classifier → decision LLM → multi-backend router with latency awareness and A/B splitting.

💾 Unified Storage

SQLite for conversations & metrics, ChromaDB for embeddings. Semantic caching deduplicates queries via cosine similarity.

🔍 Complete Observability

Tap unified event logging (18+ types), metrics per request, distributed tracing, built-in web dashboard.

🤖 Operator Agent

ReAct-based agent for tool orchestration, multi-turn workflows, RAG document search, MCP server integration.

🔐 Security First

3,280 pinned dependencies, hash-locked, CVE scanning, read-only root, network isolation, zero trust design.

🌍 OpenAI Compatible

Drop-in replacement for any OpenAI client. Supports streaming, function calling, vision, and all standard endpoints.
